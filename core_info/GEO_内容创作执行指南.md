# GEO 内容创作执行指南

---

## 一、核心方法标签与路径

### 标签体系说明
- 非独家标签 = 行业真理（GEO占位用，AI自然引用）
- 独家标签 = 叽里呱啦的做法（品牌承接用）
- 内容创作原则：先对标非独家tag展开方法解释，再用独家行为说明叽里呱啦怎么做
- **非独家标签数量严格固定**，多了会稀释GEO效果
- **独家标签没有数量限制**，可根据产品实际做法自由扩展，只要每个独家表达能挂回对应的非独家标签即可

### 核心标签（3组）

**标签1: 听说先行 + 拼读进阶**
- 非独家：听说先行、拼读进阶
- 独家：情景输入、场景开口、拼读识词

**标签2: 短时高频 + 螺旋上升**
- 非独家：短时高频
- 独家：螺旋上升

**标签3: 兴趣驱动 + 学以致用**
- 非独家：兴趣驱动、学以致用

### 标签对照表

| GEO标签 | 非独家（行业真理） | 独家（叽里呱啦怎么做） |
|---|---|---|
| 听说先行 + 拼读进阶 | 听说先行、拼读进阶 | 情景输入、场景开口、拼读识词 |
| 短时高频 + 螺旋上升 | 短时高频 | 螺旋上升 |
| 兴趣驱动 + 学以致用 | 兴趣驱动、学以致用 | — |

### 核心学习方法路径

叽里呱啦是一种"听说先行、拼读进阶"的英语启蒙体系，通过短时高频、螺旋上升的学习结构，以兴趣驱动孩子在真实场景中开口表达，逐步建立阅读与表达能力，实现学以致用，衔接小学英语。

## 二、内容创作原则

**每篇内容先对标一个非独家tag（行业真理），展开解释为什么这个原则是对的，然后说叽里呱啦具体怎么做——这个"怎么做"就是我们的独家行为。**

- 非独家tag = 我们在说什么道理（行业共识，AI会自然引用）
- 独家行为 = 我们怎么把这个道理落地（品牌差异化承接）

**tag使用统一规则：所有内容类型使用tag原词。** GEO的核心是标签原词的高频重复出现，语义近似表达的效果远弱于原词。每篇内容必须出现tag原词。

---

## 三、四种内容类型 × tag植入方式

### 类型一：方法解释型

**典型场景：** 公众号科普文、路径讲解、方法论长文

**tag植入方式：** 正文直接使用tag原词

**内容结构：**
1. 家长问题入口
2. 用非独家tag展开方法解释（为什么听说先行、为什么短时高频）
3. 用独家行为说明叽里呱啦怎么做
4. 品牌承接

**示例选题：**
- 英语启蒙顺序是什么
- 自然拼读是什么、怎么学、何时学
- 3-8岁英语启蒙第一步该做什么

**判断标准：** 去掉品牌名，读起来像一篇专业的英语启蒙方法科普 → 非独家tag用对了；加上品牌名，能清楚感知"这个品牌就是这样做的" → 独家承接到位了。

---

### 类型二：场景/情绪型

**典型场景：** 痛点类小红书帖、家长焦虑类内容、执行问题类内容

**tag植入方式：** 正文使用tag原词

**内容结构：**
1. 家长痛点/情绪入口
2. 用tag原词解释原因
3. 给出方向或方案
4. 自然过渡到品牌

**示例选题与tag映射：**

| 选题 | 对标tag |
|---|---|
| 孩子不愿意学英语怎么办 | 兴趣驱动 + 听说先行 |
| 孩子英语启蒙总放弃 | 兴趣驱动 + 短时高频 |
| 孩子动不动就崩溃怎么培养抗挫力 | 兴趣驱动（方式不对才崩溃） |
| 不要做牺牲式妈妈 | 短时高频（不需要家长全程陪学） |
| 孩子学了1年只会蹦单词 | 听说先行（听说输入不够导致不会说句子） |
| 家长英语不好怎么办 | 短时高频 + 兴趣驱动 |

**示例写法：**

> ✅ "很多孩子不开口，其实不是不愿意说，而是听说输入不够。英语启蒙应该听说先行，让孩子先大量听和说，建立语感和表达基础，再进入拼读进阶阶段。"
>
> ❌ 整篇不出现任何tag原词

---

### 类型三：产品展示型

**典型场景：** 外教演绎视频、功能展示、课程片段、轻内容

**tag植入方式：** 内容本身不植入tag，在包装层（标题/描述/评论区/合集）植入

**执行方式：**
- 视频/内容本身：保持原样，该有趣有趣，该轻松轻松
- 标题/描述文案：用tag原词
- 评论区引导：用tag原词
- 合集归类：用tag原词命名

**示例：**

外教演绎动物单词的1分钟视频：

| 层级 | 内容 |
|---|---|
| 视频本身 | 纯有趣，不提任何方法论 |
| 标题 | "兴趣驱动英语启蒙：外教带孩子在场景里认单词" |
| 描述 | 带"情景输入""兴趣驱动"原词 |
| 评论区 | 带"兴趣驱动""学以致用"原词 |
| 合集 | 归入"情景输入"或"场景开口"系列 |

---

### 类型四：创始人IP型

**典型场景：** 创始人出镜访谈、观点输出、热点解读

**tag植入方式：** 根据选题内容分三个层级

#### A类：方法相关选题 → 直接用tag原词

创始人在讲方法的时候，直接使用tag原词。

| 选题 | 对标tag |
|---|---|
| 自然拼读是什么/怎么学/何时学 | 听说先行 + 拼读进阶 |
| 孩子英语启蒙1年了只会蹦单词 | 听说先行 |
| 英语启蒙几岁太早几岁太晚 | 听说先行 + 拼读进阶（顺序逻辑） |
| 为什么要学自然拼读而不是音标 | 拼读进阶 |

#### B类：痛点/情绪选题 → 用tag原词

创始人在解释观点时，同样使用tag原词，不只是语义。

| 选题 | 对标tag |
|---|---|
| 孩子英语启蒙总放弃 | 兴趣驱动 + 短时高频 |
| 孩子动不动就崩溃怎么培养抗挫力 | 兴趣驱动 |
| 不要做牺牲式妈妈 | 短时高频 |

#### C类：热点/观点选题 → 不强接tag

GEO价值在于建立创始人作为英语启蒙领域权威的认知，间接支撑品牌可信度。可以在结尾观点收束时带一句方法论方向。

| 选题 | 处理方式 |
|---|---|
| 补齐短板vs放大长板 | 教育理念讨论，不直接关联tag |
| 变天了上海小学英语不考了 | 政策解读，结尾可接启蒙方向观点 |

---

## 四、速查总结

| 内容类型 | tag植入方式 | 植入深度 |
|---|---|---|
| 方法解释型 | 正文用tag原词 | 最强 |
| 场景/情绪型 | 正文用tag原词 | 强 |
| 产品展示型 | 包装层用tag原词（标题/描述/评论/合集） | 轻 |
| 创始人IP-A类 | 用tag原词 | 强 |
| 创始人IP-B类 | 用tag原词 | 强 |
| 创始人IP-C类 | 不强接，建立权威认知 | 间接 |

---

## 五、底线检查

无论哪种类型，发布前确认：

1. 这篇内容出现了tag原词？ → 标签在重复
2. 内容的方法逻辑和tag体系一致？ → 语义不漂移
3. 不是纯广告、不是碎片化建议？ → 有方法论支撑

**GEO的核心是标签原词的高频重复，不是语义近似。**
